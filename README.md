## ğŸ”§ Technical Decisions

### Why Supabase over Firebase?

Initially built with Firebase, we migrated to Supabase because:
- **Relational Database**: Social media apps require complex relationships between users, messages, topics, and rooms
- **Better SQL Support**: More efficient queries for social graph operations
- **Scalability**: Superior performance for real-time social features
- **Developer Experience**: Easier to manage complex data structures
